# Онлайн-библиотека
Сайт с библиотекой книг

Посмотреть на сайт без скачивания и запуска сервера можно по [этой ссылке](https://vladislavvenin.github.io/Library/pages/index1.html)
## Требования
Для запуска сервера требуется:

python3
```
pip install -r requirements.txt
```
## Запуск
Для запуска сервера используйте команду:
```
python render_website.py
```
Скрипт также поддерживает следующие аргументы:

- `-p`, `--path` - ключ для указания альтернативного пути к файлу с данными для карточек на сайте
- `-c`, `--cards` - ключ для указания кол-ва карточек на странице

далее перейдите по ссылке [`http://127.0.0.1:5500/pages/index1.html`](http://127.0.0.1:5500/pages/index1.html)
